## Chainlink (Available)

Chainlink issue tracking is available but not required. Use it when it helps.

```bash
chainlink issue quick "title" -p <priority> -l <label>   # Create + label + work
chainlink issue list -s open                              # See open issues
chainlink issue close <id>                                # Close issue
chainlink session work <id>                         # Mark focus
chainlink session end --notes "..."                 # Save handoff
```
